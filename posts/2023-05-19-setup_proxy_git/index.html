代理折腾记录 - git proxy

May 19, 2023 One-minute read
Setup
proxy • setup • git

配置 Link to heading

需要先在本地配置启动好socks代理和http代理，我用的是trojan-go和privoxy，然后

# 配置http代理
git config --global http.proxy "http://127.0.0.1:8118"
# 配置socks代理
git config --global core.gitproxy "socks5://127.0.0.1:1080"
# 查看配置
git config --global -l
